CSS란?

CSS는 Cascading Stylesheet의 약자이며 HTML같은 마크업 언어로 작성된 문서를 꾸미기 위해 사용합니다. HTML 문서의 head 요소 안에style 요소를 넣고 CSS 코드를 적으면 해당 문서를 다양하게 꾸밀 수 있습니다.

CSS에 대한 더 자세한 설명은 CSS 문서를 참고하세요.

블럭 요소 목록

다음은 HTML안에서 자주 사용되는 요소들입니다.
  1. p:단락(Paragraph)
  2. h1:큰제목(Heading1)
  3. ul:순서없는 목록(Unordered List)
  4. ol:순서있는 목록(Ordered List)
  5. table:표(table)

HTML이란?

HTML은 하이퍼텍스트 마크업 언어(Hypertext Markup Language)의 약자로, 웹 페이지를 만들 때 사용되는 문서 형식입니다.

하이퍼텍스트(Hypertext)란 하이퍼링크(Hyperlink)를 담고 있는 문서를 뜻합니다. 하이퍼링크는 문서들이 서로 연결되어 있는 고리를 말합니다.

인라인 요소 목록

다음은 HTML에서 자주 사용되는 요소들입니다.
  • a:하이퍼링크(Anchor)
  • img:이미지(Image)
  • span:일반용도의 인라인(Span)
  • strong:중요(Storng)
  • em:강조(Emphasis)
